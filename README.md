# Simple Vector Editor (CLI)

Консольное приложение для создания и управления списком геометрических фигур. Тестовое задание для компании Neiros.

## Описание реализации
В проекте использован объектно-ориентированный подход (ООП):
* **models.py**: Содержит иерархию классов фигур. Базовый класс `Shape` задает общую логику, а наследники (`Point`, `Line`, `Circle`, `Square`) реализуют специфические параметры.
* **editor.py**: Логический контроллер, который отвечает за хранение данных и управление ими (отделение логики от интерфейса).
* **main.py**: Точка входа в приложение. Реализован бесконечный цикл взаимодействия с пользователем и базовая валидация вводимых данных.


## Как запустить
1. Склонируйте репозиторий.
2. Убедитесь, что у вас установлен Python 3.8+.
3. Запустите программу:
   ```bash
   python main.py
